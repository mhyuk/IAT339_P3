Interactive Elements

In-text link

    CSS selector:

    .mainFont

    Sample code:

    <a href="#" class="mainFont">Click me</a>

    Rendered element:

    Click me

Navigation Link

    CSS selector:

    .navbar
    .navUnderline
    .active

    Sample code:

    <nav>
    <ul class="navbar">
    <li><a class="navUnderline active" href="#">HOME</a></li>
    <li><a class="navUnderline" href="#">ABOUT</a></li>
    <li><a class="navUnderline" href="#">PROJECT</a></li>
    <li><a class="navUnderline" href="#">CONTACT</a></li>
    </ul>
    </nav>

    Rendered element:

    • HOME
    • ABOUT
    • PROJECT
    • CONTACT

Submit/Reset Buttons

    CSS selector:

    .button-sub
    .mainFont

    Sample code:

    <a href="#" class="button-sub mainFont">Submit</a>

    Rendered element:

    Submit Reset

Text Input Field with Label

    CSS selector:

    .input

    Sample code:

    <input type="text" class="input" placeholder="Name">

    Rendered element:

Radio Button with Label

    CSS selector:

    .radioContainer
    .checkmark

    Sample code:

    <form>
    <label class="radioContainer">Rose Pink<input type="radio" checked="checked" name="type" value="pink">
    <span class="checkmark"></span>
    </label>
    </form>
